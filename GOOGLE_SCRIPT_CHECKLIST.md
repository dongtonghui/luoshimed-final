### 3. 部署 Web 应用（2分钟）
- [ ] 点击「部署」→「新建部署」

- [ ] 描述：网站询盘表单接口
- [ ] 执行身份：我
- [ ] 访问权限：任何人
- [ ] 点击「部署」
- [ ] 授权访问（首次需要）
- [ ] **复制生成的 Web 应用 URL**

- [ ] 打开 `contact.html`


## 🐛 常见问题速查

### 问题1：跨域错误 (CORS)
**现象**：浏览器控制台显示 CORS policy 错误
**解决**：重新部署，确保「访问权限」设为「任何人」

### 问题2：404 Not Found
**现象**：请求返回 404
**解决**：检查 URL 是否完整复制，不要有多余空格

### 问题3：数据未写入表格
**现象**：返回成功但表格为空
**解决**：
1. 检查 Apps Script 的「执行」日志
2. 确认表格是当前活动的 sheet
3. 检查字段名是否匹配

### 问题4：邮件未收到
**现象**：数据写入但无邮件通知
**解决**：
1. 检查邮箱地址是否正确
2. 查看垃圾邮件文件夹
3. Google 免费账户每日邮件配额 100 封
